# El Baúl Website

[![CI](https://github.com/ppardalj/el-baul-web/actions/workflows/deploy.yml/badge.svg)](https://github.com/ppardalj/el-baul-web/actions)

## :pick: Requirements

- [Hugo](https://gohugo.io/getting-started/installing/)
- [Hugo-Extended (scss extension)](https://gohugo.io/getting-started/installing/#chocolatey-windows)
- Make (For windows install with [choco](https://community.chocolatey.org/packages/make))

## :rocket: Build and Release

## Development server

```sh
make dev
```

## Build for production

```sh
make publish
```
