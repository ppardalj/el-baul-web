# Cómo informar de un error

Aunque ponemos mucho cuidado al desarrollar El Baúl, a veces pueden aparecer errores.

Si encuentras uno, puedes informarnos desde **Ayuda > Contactar con soporte**.

Si es posible, incluye:

- Qué estabas haciendo.
- Qué esperabas que ocurriera.
- Qué ocurrió realmente.
- Una captura de pantalla o una grabación de pantalla.

Cada incidencia nos ayuda a mejorar el producto para todas las familias que utilizan El Baúl.

Gracias por dedicar unos minutos a comunicárnosla.
