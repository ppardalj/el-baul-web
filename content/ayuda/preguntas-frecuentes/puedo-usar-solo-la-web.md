# ¿Puedo usar solo la web?

Sí.

La versión web permite realizar las tareas habituales de El Baúl, como:

- Crear baúles.
- Crear capítulos.
- Subir fotografías.
- Escribir recuerdos.
- Invitar a otras personas.

La aplicación móvil simplemente hace algunas acciones más cómodas, especialmente cuando quieres guardar fotografías directamente desde tu teléfono.
