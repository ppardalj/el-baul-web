# Crear un nuevo Baúl

Crear un baúl es el primer paso para empezar a conservar una historia.

Durante la creación solo tendrás que elegir un nombre para identificarlo.

Más adelante podrás:

- Invitar a tu familia.
- Crear capítulos.
- Subir fotografías.
- Escribir recuerdos.

Un baúl suele representar una familia, aunque también puede utilizarse para cualquier grupo estable de personas que comparta una historia común.

No hace falta configurarlo todo desde el principio.

Puedes empezar con un único capítulo e ir construyéndolo poco a poco.
