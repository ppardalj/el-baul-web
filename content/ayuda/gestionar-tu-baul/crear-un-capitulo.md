# Crear un Capítulo

Los capítulos organizan vuestra historia en acontecimientos fáciles de recordar.

Para crear uno nuevo solo tienes que elegir un título y, si lo deseas, una fecha.

Un buen capítulo suele representar un momento importante, por ejemplo:

- Un viaje.
- Una boda.
- Un cumpleaños.
- Una Navidad.
- El nacimiento de un hijo.

Después podrás añadir fotografías y recuerdos para completar esa historia.

No hace falta crear muchos capítulos.

Es preferible tener pocos capítulos bien organizados que decenas de ellos sin contexto.
